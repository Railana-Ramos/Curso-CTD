1. Quem inventou o sistema de controle de versão Git e por quê?
O Git foi criado por Linus Torvalds, que também é conhecido por ter criado o Linux.
Ele decidiu desenvolver o GIT pois ficou insatisfeito com uma mudança de licença de utilização do BitKeeper, 
um sistema de controle de versão que era utilizado por ele e sua equipe na epoca.
